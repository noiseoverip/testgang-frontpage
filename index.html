Features many more to come...

Easy and convenient test execution

Configure and run your tests on multiple devices with a couple of clicks and easy to use manaagement web app.

Supported testing Frameworks

We support test cases written with popular testing frameworks like Robotium , Calabash and UiAutomator for UI testing.

Faster parallel test execution

Improve test execution speed by running tests in FAST mode which will split and run your tests in parallel on multiple devices.

Your own testing community

Sharing devices with your colleagues has never been easier - share between groups, single users or all users.

Detailed reports

Test run reports include lists of devices on which tests were executed, number of tests run as well as logs collected from device during test execution.

Integrate and automate

Extensive REST API interface that can be used with you CI environment or to automate repetitive tasks. Check out REST API documentation.

Support multiple OS

Required client software runs on all major platforms: Linux, Windows and Mac.

How does it work ?

Connecting Android phone
Download Android SDK, download our client app, run client app and you're done. Now, your device will be shown in management web app dashboard and you will be able to run tests on it and share it with your colleagues. Client software acts as a thin agent between you, Android phone and TestGang servers.
Device sharing
Once your connected devices are visible in the dashboard you can share them with other users or user groups. This will allow others to execute tests on your device. Check client guide for more information. Access can be revoked at any time and you can unplug your device at any time.
Running tests
Before running a test you will need to upload your applications APK file as well as test case package (JAR) that was built by the chosen test framework. After this you will create a test configuration that defines the way you want to execute the tests and on which devices. TestGang will distribute your tests among available devices at a given time and you will be presented with test results and logs collected from all devices.

FAQ most important questions

How much does it cost?

Nothing, this service is completely free.

Can I share my device with anyone?

Yes! But you should not share your personal phone with your private data on it. Also, we do not recommend leaving your SIM card in it.

How many devices do you have in the pool?

We do not control the number of devices, itâ€™s completely dynamic and depends on devices shared by the members.

How is this useful for me?

In many ways. For example you and your friends/colleagues can connect devices into a shared pool and use it for running tests faster including multiple devices, or you can run a number of Android emulators on your server and use this platform to manage test execution on all of them in parallel, or you can simply share your unused devices with all other users.

Why should I test my apps?

Because there numerous variations of devices, screen sizes and other differences in Android devices that potentially can break your application or make it behave unexpectedly.

What kind of devices do you support?

This device sharing platform is device agnostic and supports any Android phone or tablet even including emulators.

Can I connect my Android emulator?

Yes, in fact, you can run multiple emulator on same computer.

Can I get the screenshots captured during test?

No yet, but it is coming soon.

Is this an open source project?

No, but we are going to open source at least the client code.

How did you came up with this idea?

The idea was born when thinking of a topic for the master thesis.

There are other services like this, how are you different?

We want to explore the fact that there are thousands of Android developers out there and all of them have at least one phone. If all of them would connect to this service, there would be no need for expensive test runs on a device cloud based services.

How many Android phones can I connect to one computer?

Probably as many as your USB bus will hold. Our client software does not limit it in any way.

Can i ask my friends to connect their phones at night so I could run tests on them?

Sure, you could ask your relatives and friends to help you out with testing by sharing their Android devices with you.

Why should I share my Android with others?

No particular reason for now, but we are thinking about introducing some tit-for-tat strategy or bonus points which would encourage to share your devices with others and in return, received for example additional time for your test runs.

Can someone harm my phone or delete my data during test execution?

Yes! We cannot possible protect you from this (although we have some ideas for the future) so be careful with whom you share your device.

How does it work ?

Connecting Android phone
You can connect your Android phone to TestGang network using our lightweight client app. By default, all your devices will be assigned to your private device pool and you will be able to run tests on them. However, this is not a neccesary step in order to make use of TestGang.
Device sharing
Your connected devices are visible in the dashboard you can share them with other users or user groups. This will allow others to execute tests on your device. You will stay in full control - access can be revoked and device can be unpluged at any time.
Running tests
To run tests: upload application and test packages, configure test parameters - choose device types, distribution mode and execute ! TestGang will distribute your tests among selected devices and you will be presented with test summary, logs, screenshots and video.

Use cases few example on how TestGang can be used...

Are you a member of a developer team, or have friends developers ?

Connect all Android's into your teams device pool. Every team member can then easily execute tests on all devices simultaneously.

Have an Android phone used for development only ?

Connect it to TestGang and share it with other users. This way to will earn testing time which can later be used to run tests on other devices !

Lack of own devices and no friends to share them with you?

No problem, you can buy testing time and test on as many devices provided by us or other users

Features many more to come...

Easy and convenient test execution

Configure and run your tests on multiple devices with a couple of clicks and easy to use manaagement web app.

Supported testing Frameworks

We support test cases written with popular testing frameworks like Robotium , Calabash and UiAutomator for UI testing.

Faster parallel test execution

Improve test execution speed by running tests in FAST mode which will split and run your tests in parallel on multiple devices.

Your own testing community

Sharing devices with your colleagues has never been easier - share between groups, single users or all users.

Detailed reports

Test run reports include lists of devices on which tests were executed, number of tests run as well as logs collected from device during test execution.

Integrate and automate

Extensive REST API interface that can be used with you CI environment or to automate repetitive tasks. Check out REST API documentation.

Support multiple OS

Required client software runs on all major platforms: Linux, Windows and Mac.

How does it work ?
  • Become a member by registering
  • Use devices shared with you to run tests for your application
  • Analyse test reports and device logs
  • Share device with your colleagues
How to share devices ?
  • Download the client
  • Connect your Android phone to your Windows/MAC/Linux laptop
  • Login to “TestGang Manager” and choose people to share your devices with.
How to run tests ?
  • Upload your Android application (APK) file and test JAR files.
  • Create configure your test
  • Run test and wait for results
Automation
Executing a test can be as simple as sending a http request like:
  curl -X PUT -u name:password -H "Content-Type: application/json" -d '{"testconfigId":"12"}' ht tp://rip.testgang.net/rip/testrun

FAQ

What is application package?

Application package or a test target is meant to be uploaded to the TestGang servers. It will be attempted to install it on maximum three devices matching predefined requirements.

What is test package?

Test package contains the actual tests to be run against defined application. Similar to application package it should uploaded onto TestGang servers for one to be able to use it.

What is test configuration?

Test configuration is used to define a link between application and test package, and the device type. Multiple device types are supported as well.

Features and many more to come...

Easy and convenient test execution

Configure and run your tests on multiple devices with a couple of clicks using easiy to use management web app.

Supported testing Frameworks

We support test cases written with popular testing frameworks like Robotium , Calabash and UiAutomator for UI testing..

Faster parallel test execution

Improve test execution speed by running tests in FAST mode which will split and run your tests in parallel on multiple devices

Your own testing community

Sharing devices with your colleagues has never been easier - share between groups, single users or all users.

Detailed reports

Test run reports include lists of devices on which tests were executed, number of run, failed test cases as well as logs collected from device during test execution.

Integrate and automate

Extensive REST API (http) interface that you can use to integrate with you CI environment or automate repetitive tasks. Check out REST API documentation.

Multiple OS support

Required client software runs on all major platforms: Linux, Windows and Mac so you should have no problems hooking it up with your PC.

How does it work ?

Connecting Android phone
Download Android SDK, download our client app, run client app and you're done. Now, your device will be shown in management web app dashboard and you will be able to run tests on it and share with your colleagues. Client software acts as a thin agent between you Android phone and TestGang servers.
Device sharing
Once your connected devices is visible in a dashboard you can share this device with other users or user groups. This will allow others to executed their tests on your device. Check client guide for more information. Access can be revoked at any time and you can unplug your device at any time.
Running tests
Before running test you will for sure need to upload your applications APK file as well as test case package (JAR) which will be built by the test framework that you are using. After this you will have to quickly select from a couple of options and create test configuration which will define the way you want run test and on which devices. TestGang will distribute your test among available devices at a given time and you will be present with test results and logs collected from all devices.
